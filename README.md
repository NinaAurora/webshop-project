Personal resume site

Animations using:
https://ianlunn.github.io/Hover/
https://animate.style/

# Installation
- cd ~/
- cd /your-sites-folder/nina-aurora-resume

- npm install
- composer install

- touch .env
- go to your code editor (Visual Studio Code etc.)
- open .env file and add: APP_NAME="My App"

Thats it ;-)
